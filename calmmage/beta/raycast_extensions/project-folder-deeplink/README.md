# Project Folder Deeplink

Open a project folder with a specified tool

Similar to running the following in the console:
```bash
open -a "Visual Studio Code" /path/to/folder
```

Commands list:
- open-dir - open the path provided via Raycast in the selected tool
- open-dir-clipboard - open the path from clipboard in the selected tool
- get-dir-link - get a raycast deep link for the path in the clipboard in the selected tool
- sample-command - open hardcoded path in the selected tool
